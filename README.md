<details>
<summary><strong>Week 3 ‚Äî Collections & Generics</strong></summary>

<details>
<summary><strong>Week 4 ‚Äî Delegates & Events</strong></summary>

- Implement a simple **Observer Pattern** example.

</details>

<details>
<summary><strong>Week 5 ‚Äî Async & Await</strong></summary>

- Practice with async file operations or HTTP requests.

</details>

<details>
